## 📚 교과목 개요
### 수업 개요 및 목적
컴퓨터로 문제를 해결하기 위해서 가장 기초가 되는 자료구조인 **배열, 리스트, 스택, 큐, 그래프, 트리 및 기본적인 탐색, 정렬 알고리즘**을 학습한다.  
또한 이를 C언어로 구현하는 법을 배운다.

### 수업 목표
컴퓨터로 문제를 효율적으로 해결하기 위한 방법과 기술을 습득하는 것을 목표로 한다.
